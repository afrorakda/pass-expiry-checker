# Pass-Expiry Checker
A hyper-minimalist "freshness tracker" for your passwords. 
This tool never asks for your actual password—it only tracks how long you've been using it.

## 🚀 Concept: Security is Perishable
Pass-expiry is the ultimate "hidden thorn." We all know we should change passwords regularly, but without a clear visualization of how "old" they are, we tend to ignore the risk. This tool turns that vague anxiety into a concrete number of days.

## ✨ Features
- **Zero Privacy Risk:** Tracks only Service Names and Dates. No passwords, no clouds, no tracking.
- **Freshness Visualization:** Instantly categorizes passwords by age:
  - **Healthy:** Less than a year old.
  - **Old:** Over 1 year (needs attention).
  - **Critical:** Over 2 years (high risk).
- **Auto-Sorting:** Automatically places the "oldest" (riskiest) passwords at the top of the list.
- **Backup & Restore:** Export your list to a JSON file and import it anytime. 100+ items can be restored in a second.
- **UI Design:** 15px rounded corners, minimalist white-and-black theme, optimized for mobile Safari/Chrome.

## 🛠 How to Use
1. **Label:** Enter the name of the service (e.g., "Amazon").
2. **Date:** Select the day you last changed that password.
3. **Check:** Hit "Track Freshness" to add it to your list.
4. **Action:** If an item turns red (Critical), it’s time to head to that site, change your password, and update the date here.

## 📐 The Philosophy
This is not a password manager; it is a **Security Mirror**. It reflects the reality of your digital hygiene. By using this alongside a generator like [#004 Password Gen], you can maintain a high-security lifestyle without the friction of complex enterprise software.
